Лабораторная работа №1
Описание работы
В ходе выполнения лабораторной работы я создал простую HTML-страницу-визитку, используя HTML и CSS. Визитка включает информацию о себе, список хобби и контактные данные. Я также постаралась создать современный и минималистичный дизайн в сиреневых тонах с круглым фото и плавными переходами.

Ответы на вопросы:

Что удалось сделать легко?
Создание структуры HTML-страницы и оформление стилизации с использованием CSS оказалось достаточно простым. Я смогла легко разместить основные элементы: заголовок, описание обо мне, хобби и контакты. Также не составило труда задать стили для фона, шрифтов и блоков.

Где помог ИИ?
ИИ был полезен в процессе дизайна: предложил использовать сиреневые тона для оформления, улучшить стиль с помощью плавных анимаций, а также подсказал, как сделать изображение круглым, несмотря на то, что оно было прямоугольным изначально. Также ИИ посоветовал добавить медиа-запросы для мобильной версии сайта, чтобы визитка корректно отображалась на разных устройствах.

Где пришлось разбираться самому?
При создании визуально привлекательного дизайна пришлось немного поэкспериментировать с цветами и шрифтами, чтобы добиться нужного эффекта. Например, я выбрала шрифт для заголовков и текста, который подходит под минималистичный стиль.
